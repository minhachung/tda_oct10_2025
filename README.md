# tda_oct10_2025

Topological Data Analysis of the October 10, 2025 cryptocurrency liquidation cascade. This project applies persistent homology and related TDA techniques to high-frequency market data spanning the October 10, 2025 event in order to characterize the topological structure of the cascade, test for trend and regime-change signatures against null models, and evaluate the diagnostic value of TDA-derived indicators relative to conventional market microstructure measures.
